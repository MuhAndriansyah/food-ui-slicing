# food-ui-slicing


## Membutuhkan Extension
  - Live Server (VSCODE)
  
Install SASS secara global

```
npm install -g sass
```


### Cara Compile SASS pada project ini

```
npm run sass
```

### Cara Jalankan LiveServer

1. Klik kanan pada index.html
2. Open with Live Server
